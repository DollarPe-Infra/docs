In the meantime, you can use our [REST API](/api-reference/overview/introduction) directly with any Python HTTP client like `requests`:

```python
import requests

API_KEY = "your-api-key"
API_SECRET = "your-api-secret"

# Example request to fetch exchange rate
response = requests.post(
    "https://sandbox.dollarpe.xyz/pos/api/v1/payout/fetch-rate",
    headers={
        "X-API-KEY": API_KEY,
        # Add other required headers
    },
    json={
        "asset": "USDT",
        "network": "BSC",
        "amount": 100.5
    }
)

print(response.json())
```
